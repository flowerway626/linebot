## **人氣排行 / 最新排行**<br>

  - 整理出 TOP12 影劇，最新排行操作方式同理<br>
  或是直接輸入 **國家名 + 最新 or 人氣**，也可觸發事件<br>
  <img src="https://i.imgur.com/Nqd9z5F.jpg" width="300" >
  <br>
